# JavaScript
- [Pokemon](https://yu1129.github.io/JavaScript/Hw2_Pokemon.html)
    - 使用技術 :
        - 使用 `onclick` 事件監聽，簡單又快速的綁定事件

- [iPadAir](https://yu1129.github.io/JavaScript/Hw6_iPadAir.html)
    - 使用技術 :
